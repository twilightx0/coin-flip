# Coin Flip Arena

A simple and interactive coin flip web application with a clean interface and animated coin flips.

## 🌐 Live Demo

https://coin-flip0.netlify.app/

## ✨ Features

- Animated coin flip
- Responsive interface
- Simple and lightweight
- Fast loading

## 🛠️ Built With

- HTML
- CSS
- JavaScript

## 🚀 Running Locally

1. Clone the repository

```bash
git clone https://github.com/YourUsername/YourRepository.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
